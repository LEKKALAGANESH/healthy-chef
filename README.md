# HealthyChef — Invoice Generator

MERN take-home for HealthyChef (Apr 2026).

## Quickstart
1. `npm install`
2. Copy `server/.env.example` → `server/.env`, fill `MONGO_URI`
3. Copy `client/.env.example` → `client/.env`
4. `npm run dev`  — client on 5173, server on 5000

## Stack
React 18 + TS (Vite) · Express + TS · MongoDB Atlas · styled-components · TanStack Query · Redux Toolkit · @react-pdf/renderer
