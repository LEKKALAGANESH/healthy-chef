# Screenshots

Place PNG or JPG exports in this folder using the exact filenames referenced by the root `README.md`:

| Filename | Content |
|---|---|
| `items.png` | Items catalogue page with a few rows + the create/edit modal open |
| `new-invoice.png` | New Invoice page with customer details filled in and 2–3 line items showing live totals |
| `invoice-pdf.png` | A page from the downloaded PDF (header, parties block, items table, summary) |
| `history.png` | History dashboard with at least one row expanded |

## Capture tips

- Use a 16:9 or 16:10 viewport (1280×800 or 1440×900) for consistent framing.
- Hide browser dev-tools, bookmarks bar, and notifications.
- Prefer `.png` for UI screenshots; PDF page captures can be `.jpg` to stay small.
- Keep each image under ~500 KB — compress with [squoosh.app](https://squoosh.app) if needed.
